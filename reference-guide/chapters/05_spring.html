Spring Integration

Use in like Spring bean for creating Action-Handler API instead of classic API. You can use features like context objects in your Engine and inject them to handlers.
Example of Engine's bean in Spring config file: 

<beans ...>

	<bean id="someEngine" class="com.gf.core.Engine">
		<property name="scanAndPut">
			<list>
				<value>foo.blah</value>
			</list>
		</property>
		<property name="contextObjects">
			<list>
				<ref bean="someBean"/>
				<ref bean="otherBean"/>
			</list>
		</property>
		<property name="filterTypes">
			<list>
				<value>bar.baz.SomeFilter</value>
			</list>
		</property>
		<property name="interceptorTypes">
			<list>
				<value>bar.baz.SomeInterceptor</value>
			</list>
		</property>
	</bean>

</beans>
